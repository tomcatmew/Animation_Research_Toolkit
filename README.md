# Research_Assignment_1
build the GLFW before build the assignment code
```
cmake . -A x64
cmake --build . --config Release
```

## Screenshot for One_Triangle Assignment
Build the cmake file in one_triangle folder and run in Debug mode
```
cmake -S . -B build
```
